# 📊 Student Performance Analysis

This project analyzes student data using Python, Pandas, and Matplotlib to generate meaningful insights.

---

## 🚀 Features

- ✅ Pass/Fail classification of students
- 📊 Average marks calculation
- 🏙️ City-wise performance analysis
- 🥇 Topper and lowest scorer identification
- 📈 Data visualization using bar graphs

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

## 📂 Dataset

The dataset contains:
- Name of student
- Marks
- City

---

## 📈 Insights Generated

- Identified top-performing student
- Compared city-wise performance
- Calculated overall and category-wise averages

---

## ▶️ How to Run

1. Install required libraries: pip install pandas matplotlib 
2. Run the script:python analysis.py
